Создание видео ролика для ZS GMX + GS.

Перетащите мышкой видео файл на vid2zx.bat.
После обработки в той же папке, где был видео файл, должна появиться папка с готовым видео и плеером.
Скопируйте результат на диск SMUC или Z Controller.

На диске FAT32 должно быть достаточно свободного места, чтобы файл не получился фрагментированным.
Размер кластера только 4096.
