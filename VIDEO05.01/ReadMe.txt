Создание видео ролика для ZS GMX + GS/Covox.

Перетащите мышкой видео файл на vid2zx.bat.
После обработки в той же папке, где был видео файл, должна появиться папка с готовым видео и плеером.
Скопируйте результат на диск SMUC или Z Controller.

На диске FAT32 должно быть достаточно свободного места, чтобы файл не получился фрагментированным.


Можно регулировать скорость звука в параметре
ffmpeg ... -ar 22500 ...
где 22500 - битрейт
чем больше битрейт, тем медленнее звук.

Качество картинки в параметрах 
DaDither.exe GMX160 ... /p FitResultInSource /dm Floyd-Steinberg /ed 80 /fc 000000 /lf 10
где
Первый параметр либо GMX640, либо GMX160
p - position
dm - dithering method
ed - error diffusion
fc RRGGBB - hex значения цветов
lf - Lightness fix

Можно создавать другие форматы видео. Подробнее в инструкции к плееру.
